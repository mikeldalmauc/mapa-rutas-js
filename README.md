# mapa-js

Este proyecto es una aplicación JavaScript sencilla que utiliza archivos de texto para mostrar rutas y formas en un mapa.

## Estructura del proyecto

- `app.js`: Lógica principal de la aplicación.
- `index.html`: Página principal de la aplicación web.
- `routes.txt`: Archivo de texto con información de rutas.
- `shapes.txt`: Archivo de texto con información de formas.

## Cómo ejecutar

1. Abre `index.html` en tu navegador web preferido.
2. Asegúrate de que los archivos `routes.txt` y `shapes.txt` estén en el mismo directorio que `index.html` y `app.js`.

## Descripción de archivos

- **app.js**: Contiene el código JavaScript que carga y procesa los archivos de rutas y formas.
- **index.html**: Estructura HTML de la aplicación, enlaza con `app.js`.
- **routes.txt**: Define las rutas a mostrar (formato específico según implementación).
- **shapes.txt**: Define las formas a mostrar (formato específico según implementación).

## Personalización

Puedes modificar los archivos `routes.txt` y `shapes.txt` para cambiar las rutas y formas que se muestran en el mapa.
